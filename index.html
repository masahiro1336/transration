クイックスタート

01
翻訳する
原文を入力すると右に訳文が自動表示。言語ペアを変えると履歴・Word Book・クイズが自動連動します。整形ボタンで長文を文ごとに分割できます。
02
保存する
フッターの保存先セレクトで History または Word Book を選んで保存アイコンをタップ。言語ペアと紐付いて整理されます。
03
使う・復習する
カードをクリックで発音、ダブルクリックで内部検索。ループ再生・クイズ・Word Dropで繰り返し定着させます。

音声機能

💡
原文・訳文それぞれで音声エンジンを個別に設定できます。フッターのエンジン名をクリックするとサイクル切替。
対応エンジン
  • ブラウザ — ネイティブ SpeechSynthesis
  • ElevenLabs — API キー設定で利用可
  • VOICEVOX — localhost:50021
  • Style-Bert-VITS2 — localhost:5000
再生モード
  • 全文 — テキストエリア全体を読み上げ
  • 選択 — 選択範囲のみ再生
  • ループ — 原文 / 訳語 / 交互の 3 モード
  • 選択ループ — チェックした項目のみ

プロダクトの目的

Kotoba は、翻訳・保存・整理・復習が 1 ページの中で自然につながることを設計の中心に置いています。単なる翻訳ツールではなく、入力した語句が蓄積され、あとから再利用・再学習できるワークスペースとして機能します。

解決していること
  • 翻訳結果がその場限りで消える
  • 聞く・覚える・調べるが分断される
  • 単語帳化と復習が手間になる
向いている用途
  • 学習 — 単語・発音・訳語の反復
  • 調査 — 記事を読みながら蓄積
  • 実務 — 下訳・表現整理
設計の優先順位
  • 保存・整理を 1 ページに集約
  • 言語ペアで自動連動
  • すべてローカル保存

データとプライバシー

🔒
すべてのデータはブラウザ内（IndexedDB）にのみ保存されます。外部サーバーへの送信は行いません。ブラウザのストレージ削除・別端末利用時はデータが失われます。Connection から JSON で書き出してバックアップすることを推奨します。
⚠️
翻訳リクエストは Google Translate API を経由します。契約・法務・医療・申請・公的提出などの高リスク用途では、原文確認と専門家による再検証を行ってください。
