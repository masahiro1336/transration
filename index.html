Kotoba — 使いかた

翻訳して終わらず、残して、育てて、もう一度使うための workspace です。

対象翻訳学習・語彙整理・長文読解
保存ブラウザ IndexedDB（ローカル・外部送信なし）
フロー 翻訳→ 保存→ 整理→ 復習
⚡

クイックスタート

1翻訳する

原文を入力すると右に訳文が表示されます。言語ペアを変えると履歴・Word Book・クイズが自動連動します。

2保存先を選んで残す

フッターの保存先セレクトで 翻訳履歴 または Word Book を選んでフロッピーアイコンを押します。

3クリック・ダブルクリック

カードはクリックで発音、ダブルクリックで内部検索、右クリックでインライン編集できます。

音声機能

エンジンの切り替え

原文・訳文それぞれでブラウザ・ElevenLabs・VOICEVOX・Style-Bert-VITS2 を個別に選択できます。フッターのエンジン名ボタンをクリックするとサイクル切替。

全文 / 選択範囲

読み上げボタン横の「全文」ピルをクリックすると「選択」モードに切替。テキスト選択中に読み上げると選択範囲のみ再生します。

📌

保持パッド

💡

全画面ボタン左の格子アイコンをクリックして開くフローティングメモパッドです。ウィンドウはドラッグで移動できます。

  • ドラッグで追加 — 原文・訳文エリアをドラッグしてパッドにドロップすると内容が追加されます。
  • テキスト入力・ペースト — パッドに直接テキストをタイプまたはペーストできます。
  • ダブルクリックで検索 — 語句を選択してダブルクリックすると内蔵検索が開きます。
  • 原文に送る — パッドの内容を翻訳パネルに転送して自動翻訳を起動します。
  • IndexedDB 保存 — 入力のたびに自動保存。次回起動時も復元されます。
✍️

整形機能

原文フッターの横線アイコンをクリックすると、。 ． ！ ？ … ! ? で文を区切り、文と文の間に空白行を挿入します。長文ペースト時に特に有効です。整形後は自動で翻訳が再実行されます。

🔍

内蔵検索ウィンドウ

開き方

翻訳パネル・履歴・Word Book・クイズカードでテキストをダブルクリックすると開きます。

検索モード

Google AI・通常検索・画像検索・動画検索 の4モードを切り替えられます。

このサイトについて

Kotoba は、翻訳・保存・整理・復習が1ページの中で自然につながることを設計の中心に置いています。単なる翻訳ツールではなく、入力した語句が蓄積され、あとから再利用・再学習できるワークスペースとして機能します。

解決していること
  • 翻訳結果がその場限りで消える
  • 聞く・覚える・調べるが分断される
  • 単語帳化と復習が手間になる
向いている用途
  • 学習 — 単語・発音・訳語の反復
  • 調査 — 記事を読みながら語句を蓄積
  • 実務 — 下訳・表現整理
設計の優先順位
  • 保存・整理を1ページに集約
  • 言語ペアで自動連動
  • すべてローカル保存

利用上の注意

💾

データはすべてブラウザ内に保存されます

IndexedDB を使用し、外部サーバーへの送信は行いません。ブラウザのストレージ削除・別端末への移行時はデータが失われます。翻訳リクエストは Google Translate API を経由します。

⚠️

翻訳・音声・外部検索は外部サービスの影響で結果が変わることがあります。契約・法務・医療・申請・公的提出・研究成果確定などの高リスク用途では、必ず原文確認と専門家による再検証を行ってください。
